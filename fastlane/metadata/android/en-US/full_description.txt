Agora is a fully open-source, BYOK (Bring Your Own Key) Android LLM client built for power users. It connects directly to 8+ AI providers — OpenAI, Anthropic, Google Gemini, DeepSeek, Qwen, OpenRouter, Ollama, and local GGUF models via llama.cpp — with no intermediary servers or tracking. Custom providers with arbitrary base URLs and API keys are supported.

Features:
* Multi-round tool calling: web search (Brave, Serper, Tavily, SearXNG), code execution, remote shell execution, memory management
* Non-linear branching: edit any past message and explore alternative conversation paths without losing context — powered by a tree-structured Room database
* On-device LLM inference via llama.cpp (GGUF) and local embeddings for RAG-powered semantic search across conversation history
* Deep reasoning model support: OpenAI o1/o3, Anthropic extended thinking, Gemini thinking, DeepSeek-R1, Qwen QwQ with streaming think-tag rendering
* Active memory and saved memory files for persistent context across conversations
* Context window management with real-time token counting and sliding window
* .agora export/import (conversations, memories, prompts, settings, keys) and third-party import from Claude and ChatGPT exports
* Markdown rendering with syntax highlighting, LaTeX math, image and file attachments
* English and Chinese language support
