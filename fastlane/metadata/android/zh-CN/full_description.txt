Agora 是一款完全开源、BYOK（自带密钥）的 Android LLM 客户端，面向高级用户。它直接连接 8+ AI 供应商，无中间服务器或跟踪，所有对话均存储在本地。

功能特性：
* 多轮智能体工具调用：网页搜索（Brave、Serper、Tavily、SearXNG）、代码执行、远程 Shell、文件操作（读写/编辑/文件搜索）、记忆管理、语义对话搜索
* 远程设备控制（Conch 协议）：端到端加密（ECDH + AES-256-GCM + HMAC-SHA256），支持多设备配置和 Claude Desktop MCP 集成
* 非线性分支对话：可编辑任意历史消息并探索替代对话路径，基于树形 Room 数据库
* 本地 LLM 推理（llama.cpp / GGUF）及本地嵌入的 RAG 语义搜索
* 深度推理模型支持：主流推理模型（OpenAI、Anthropic、Google、DeepSeek、Qwen），支持实时思考过程流式渲染
* 活跃记忆和保存的记忆文件，实现跨对话持久化上下文
* 上下文窗口管理，实时 token 计数和滑动窗口
* .agora 导出/导入（对话、记忆、提示词、设置、密钥），支持 Claude 和 ChatGPT 的导入
* Markdown 渲染，支持语法高亮、LaTeX 数学公式、图片和文件附件
* 中英文双语支持
